# Resume Maker

A modern resume builder built with **React, Vite, and Tailwind CSS**.

## Features
- Live resume preview
- Reusable component architecture
- Clean, industry-standard frontend setup
- Git versioned step-by-step development

## Tech Stack
- React (Vite)
- Tailwind CSS
- Context API
- Git & GitHub
